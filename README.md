## Description

Etherpad/Airtable Integration

## Installation

```bash
$ npm install
```

## Running the app

In src/environments, create an "environment.ts" file based on "environment.template.ts" and fill your info to make it work.

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
